# LINE-DL-Web

LINEスタンプをダウンロードできます。

1.ライブラリインストール
```
pip install fastapi
pip install requests
pip install beautifulsoup4

```

2.main.py 起動

3.コンソールに表示されたURLにアクセス

4.LINEスタンプのURLを張り付けて検索
